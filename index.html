OmniOne OpenDID Orchestrator Prerequisites

Before you begin setting up Hyperledger Fabric, you need to install the required dependencies. The following software components must be installed on your machine:

1. Operating System

Hyperledger Fabric has been tested on:

  • Ubuntu 20.04 LTS or later
  • macOS Big Sur or later
  • Windows 10 with WSL 2

2. Required Software

Ensure you have the following software installed:

  • Docker: Version 20.10 or later
  • Docker Compose: Version 1.29 or later
  • npm: Version 6 or later
  • cURL & Git: Latest versions recommended

3. Installing Docker

You can install Docker using the following command: 

curl -fsSL https://get.docker.com | bash

After installation, verify Docker is running: 

docker --version

4. Installing Go

Install Go using the command: 

wget https://golang.org/dl/go1.17.linux-amd64.tar.gz

Verify Go installation: 

go version

5. Setting Up Hyperledger Fabric

Once all prerequisites are installed, you can proceed to set up Hyperledger Fabric by downloading the necessary binaries: 

curl -sSL https://bit.ly/2ysbOFE | bash -s

This will download the required Fabric binaries and Docker images.
